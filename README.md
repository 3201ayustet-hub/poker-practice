# Poker Practice GTO

iPhoneのGitHubだけで運用しやすい、黒基調のGTO風画像検索アプリです。

## ファイル構成

- index.html
- admin.html
- data.js

## 使い方

1. GitHubに3ファイルを置く
2. GitHub Pagesで公開する
3. admin.htmlで画像を登録する
4. data.js出力をコピーする
5. GitHub上のdata.jsを丸ごと貼り替える

## 仕様

- 参加人数 / スタック / ポジション / アクションで検索
- タイトルは自動生成
- 画像は開発者ページから選択
- 画像は圧縮され、data.jsにBase64で保存
