Firestore Rules（最初のテスト用）

Firebase Console
→ Firestore Database
→ ルール
に貼り付けて「公開」してください。

注意：
このルールはURLを知っている人なら誰でも読み書きできます。
4人だけで使う前提の簡易版です。

rules_version = '2';

service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
